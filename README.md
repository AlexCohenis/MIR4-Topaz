**Official Public Topaz Source** — The most advanced open-source MIR4 (Legend of Mir 4) private server.

**Topaz Mir4** is a clean, rebuilt implementation of the official 2019-2021 server code, designed for maximum compatibility and easy community hosting.

### Features

- **Full Server Emulation** — Login, Game, API Servers
- **MySQL Database** support
- **JSON Config System**
- **Multi-language** support (EN / KR / others)
- **Modern Launcher** with auto-update
- **Anti-cheat** foundation
- **Easy one-click setup** scripts
- **Active Development** & Community Driven
